## 📌 기능

클럽 구성원이 "오늘 뭐 코딩하지?"라는 고민 없이 재미있고 신선한 코딩 아이디어를 받을 수 있도록 도와줍니다.

### 💬 명령어

```
/vibeidea [분야] [난이도] [제한시간]
```

#### 매개변수

- **분야** (선택): 웹, CLI, AI, 유틸, 게임, 기타
- **난이도** (필수): 쉬움, 중간, 어려움, 랜덤
- **제한시간** (선택): 30m, 1h, 3h, 상관없음
